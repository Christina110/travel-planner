# 旅行规划助手 - Travel Planner

一个沉浸式探索风格的旅行规划应用，帮助用户探索中国31个城市。

## ✨ 特色功能

- 🌍 **城市探索地图** - 31座城市的精美探索卡片
- 📅 **智能行程规划** - AI生成个性化旅行路线
- 🌤️ **出行天气** - 实时查询城市天气信息
- 💰 **预算管理** - 旅行开销追踪与统计
- 🗺️ **离线地图** - 景点分布与游览路线展示

## 🎨 设计风格

- **沉浸式探索风** - 电影级旅行摄影风格
- **深色主题** - 深夜蓝背景，科技青强调色
- **玻璃拟态** - 半透明卡片，模糊背景效果
- **高端科技感** - 国家地理探索杂志质感

## 🛠️ 技术栈

- React 18
- Tailwind CSS 3
- Chart.js (图表)
- Leaflet (地图)
- Babel (JSX编译)

## 🚀 快速开始

### 安装 Git

1. 下载 Git: [https://git-scm.com/download/win](https://git-scm.com/download/win)
2. 运行安装程序，选择默认选项即可

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/你的用户名/仓库名称.git

# 进入项目目录
cd travel

# 使用本地服务器运行
# 方法1: Python
python -m http.server 8000

# 方法2: Node.js
npx serve

# 在浏览器中打开
http://localhost:8000
```

## 📱 功能模块

1. **首页** - Hero区域 + 城市探索卡片 + 功能入口
2. **城市详情** - 景点推荐、美食、住宿、打卡机位
3. **行程规划** - 智能生成旅行路线
4. **天气查询** - 实时天气信息
5. **预算管理** - 开销记录与统计
6. **景点地图** - 景点分布与路线
