## 🎬 沉浸式探索体验

![旅行规划助手](https://images.unsplash.com/photo-1464822759023-fed622ff2c3b?w=1600&q=80)

### 为什么选择我们？

- **🎨 电影级视觉设计** - 深夜蓝沉浸式主题，国家地理杂志质感
- **🤖 AI智能规划** - 智能生成个性化旅行路线，让每一次旅行都独一无二
- **🌍 覆盖31座城市** - 从繁华都市到秘境小镇，应有尽有
- **💎 玻璃拟态UI** - 半透明卡片设计，科技感与美感完美融合

---

## ✨ 核心功能

### 🏙️ 城市探索地图
- 31座城市的精美探索卡片
- 电影级摄影背景，每张卡片都是艺术品
- 悬停发光效果，沉浸式交互体验

### 📅 智能行程规划
- AI生成个性化旅行路线
- 支持多日行程规划
- 自动推荐最佳游览顺序

### 🌤️ 出行天气
- 手动选择城市查询天气
- 7日天气预报
- 天气变化提醒

### 💰 预算管理
- 旅行开销追踪与统计
- 预算进度可视化
- 消费分类管理

### 🗺️ 离线地图
- 景点分布展示
- 游览路线规划
- 无网络也能畅游

## 🎨 设计亮点

| 特色 | 描述 |
|------|------|
| **深夜蓝主题** | `#071A2B` 深邃夜空般的沉浸感 |
| **科技青强调** | `#00D9C5` 未来感科技光芒 |
| **玻璃拟态** | 半透明卡片 + 模糊背景 + 微光边框 |
| **电影级摄影** | 每张城市图片都是精心挑选的旅行大片 |
| **流畅动画** | 悬停效果、渐变过渡、视差滚动 |

## 🚀 快速开始

### 方法一：直接在浏览器中打开

```bash
# 克隆仓库
git clone https://github.com/Christina110/travel-planner.git

# 进入项目目录
cd travel-planner

# 启动本地服务器（任选其一）

# Python 用户
python -m http.server 8000

# Node.js 用户
npx serve

# PHP 用户
php -S localhost:8000
```

### 方法二：在线预览

访问 GitHub Pages（即将上线）

## 🛠️ 技术栈

- **前端框架**: React 18
- **样式引擎**: Tailwind CSS 3
- **图表库**: Chart.js
- **地图组件**: Leaflet
- **图标库**: Material Icons
- **编译器**: Babel
